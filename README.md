# CPHOS LaTeX 模板
本仓库包含 CPHOS 物理竞赛联考的 LaTeX 模板，供编写试题、答案、答题卡使用。

## 目录结构
```
CPHOS-LaTeX-Template/
├── Assets/                  # 资源文件夹
├── Common/                  # 公共样式文件夹
│   └── cphostheory.sty            # CPHOS LaTeX 理论宏包文件
│   └── cphosexperiment.sty     # CPHOS LaTeX 实验宏包文件
├── Theory/                  # 理论试题与答案模板
│   ├── README.md            # 理论试题模板说明文件
│   ├── theory.tex           # 理论试题模板主文件
│   ├── theory-answer.tex    # 理论答案模板主文件
│   ├── theory-answer-card.tex # 理论答题卡模板主文件
│   ├── Problems/            # 理论试题示例文件
│   └── Answers/             # 理论答案示例文件
├── Experiment/              # 实验试题与答案模板
│   ├── README.md            # 实验试题模板说明文件
│   ├── experiment.tex       # 实验试题模板主文件
│   ├── experiment-answer.tex# 实验答案模板主文件
│   ├── experiment-answer-card.tex # 实验答题卡模板主文件
│   ├── Problems/            # 实验试题示例文件
│   └── Answers/             # 实验答案示例文件
└── README.md                # 本说明文件
```
